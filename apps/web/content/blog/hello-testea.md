테스티아(Testea)는 **무료로 쓸 수 있는 QA 도구**입니다. 테스트 케이스 작성, 테스트
스위트 구조화, 테스트 실행 추적, 마일스톤 관리를 한 곳에서 처리할 수 있도록 설계했습니다.

## 왜 만들었나요

작은 팀에서 QA 프로세스를 시작할 때 가장 흔히 마주치는 문제는, 도구가 너무 비싸거나
너무 복잡해서 도입 자체가 미뤄진다는 점이었습니다. 테스티아는 다음 세 가지를 우선했습니다.

- **즉시 시작 가능**: 회원 가입 없이 프로젝트 식별번호로 바로 접근
- **읽기 쉬운 구조**: 케이스, 스위트, 런, 마일스톤이 깔끔하게 분리
- **국문 친화**: UI 와 문서를 한국어 중심으로 작성

## 핵심 기능

| 기능          | 설명                                                 |
| ------------- | ---------------------------------------------------- |
| 테스트 케이스 | 제목 · 사전 조건 · 스텝 · 기대 결과를 표 형태로 작성 |
| 테스트 스위트 | 케이스를 도메인·시나리오 단위로 묶음                 |
| 테스트 실행   | 스위트 · 케이스 단위로 통과 / 실패 / 차단 기록       |
| 마일스톤      | 릴리스 단위 진행률과 결과 추적                       |

## 이어지는 글

다음 글에서는 효과적인 테스트 케이스를 작성하는 실무 패턴과, 테스티아의 스위트
구조화 전략을 다룹니다.
