# 대시보드

프로젝트의 테스트 현황을 한눈에 파악하고 빠르게 작업을 시작할 수 있는 대시보드 사용법을 안내합니다.

---

## 대시보드란?

대시보드는 프로젝트에 접속했을 때 가장 먼저 보이는 화면입니다. 테스트 진행 상황, 프로젝트 정보, 최근 활동 등을 요약하여 보여주며, 주요 기능으로 빠르게 이동할 수 있는 시작점 역할을 합니다.

### 주요 역할

- **테스트 현황 요약**: KPI 카드로 핵심 지표를 한눈에 확인
- **시각적 분석**: 차트로 테스트 상태 분포와 마일스톤 진행률 파악
- **빠른 접근**: 테스트 케이스, 스위트, 실행을 바로 생성하거나 조회
- **프로젝트 관리**: 저장 용량, 최근 활동 등 프로젝트 정보 확인

---

## KPI 카드

대시보드 상단에 핵심 지표를 요약한 카드가 표시됩니다.

| 카드 | 설명 |
|------|------|
| **전체 케이스** | 프로젝트에 등록된 총 테스트 케이스 수 |
| **테스트 스위트** | 생성된 테스트 스위트 수 |
| **통과율 (Pass Rate)** | 실행된 테스트 중 통과한 비율 |
| **주요 이슈** | 실패(Fail) + 차단(Blocked) 케이스 수 |
| **미실행** | 아직 실행되지 않은 테스트 케이스 수 |

### 통과율 계산 방식

```
통과율 = Pass / (Pass + Fail + Blocked) × 100
```

- 실행된 테스트가 없으면 `N/A`로 표시됩니다

### 통과율 색상 기준

| 통과율 | 색상 | 의미 |
|--------|------|------|
| 80% 이상 | 녹색 | 양호 |
| 50~79% | 주황색 | 주의 필요 |
| 50% 미만 | 빨간색 | 위험 |
| 데이터 없음 | 회색 | 미실행 |

---

## 프로젝트 정보 카드

대시보드 중간에 프로젝트 관련 정보를 보여주는 카드 영역이 있습니다.

### 프로젝트 정보

- 프로젝트 이름 표시
- **링크 복사** 버튼으로 프로젝트 URL을 클립보드에 복사
- 프로젝트 생성일 표시

### 저장 용량

- 현재 사용량 / 최대 용량 표시
- 프로그레스 바로 사용량 시각화

| 사용률 | 색상 | 의미 |
|--------|------|------|
| 80% 미만 | 녹색 | 여유 |
| 80~94% | 주황색 | 주의 |
| 95% 이상 | 빨간색 | 용량 부족 |

### 최근 활동

최근 프로젝트에서 수행된 활동을 시간순으로 표시합니다.

- 최대 7개 항목 표시
- 테스트 케이스 생성, 스위트 생성, 테스트 실행 완료 등의 활동 기록
- 상대 시간 표시 (예: "5분 전", "1시간 전")

---

## 테스트 상태 차트

테스트 실행 결과를 원형 차트(Pie Chart)로 시각화합니다.

### 표시 항목

| 상태 | 색상 | 설명 |
|------|------|------|
| **Passed** | 녹색 | 테스트 통과 |
| **Failed** | 빨간색 | 테스트 실패 |
| **Blocked** | 주황색 | 실행 불가 (차단) |
| **Not Run** | 회색 | 미실행 |

### 차트 정보

- 각 상태별 건수와 백분율 표시
- **완료율**: 실행된 테스트 비율
  - `완료율 = (Pass + Fail + Blocked) / 전체 × 100`
- 전체 테스트 수 배지 표시

### 테스트 실행 선택

차트 상단의 드롭다운에서 특정 테스트 실행을 선택하여 해당 실행의 결과만 볼 수 있습니다.

- 진행 중(In Progress) 실행이 우선 선택됩니다
- 실행 이름, 케이스 수, 완료율이 함께 표시됩니다

> **참고**: 테스트 실행이 없으면 차트 대신 **"테스트 실행 하기"** 버튼이 표시됩니다.

---

## 마일스톤 간트 차트

마일스톤과 소속 스위트의 일정과 진행률을 타임라인으로 보여줍니다.

### 차트 구성

- **마일스톤 행**: 상위 항목으로 전체 진행률 표시
- **스위트 행**: 마일스톤 하위에 들여쓰기로 표시
- **프로그레스 바**: 각 항목의 테스트 완료 비율

### 타임라인 탐색

- 한 번에 6주 단위로 표시
- **이전/다음** 버튼으로 기간 이동
- **오늘 표시선**: 현재 날짜에 세로선 표시
- 주간 단위 눈금선 표시

### 테스트 실행 필터

- 드롭다운에서 특정 테스트 실행을 선택하면 해당 실행 기준의 마일스톤 데이터를 표시합니다
- 마일스톤 행을 접거나 펼쳐서 스위트를 표시/숨김할 수 있습니다

---

## 테스트 케이스 섹션

대시보드 하단에 최근 테스트 케이스를 간략히 보여줍니다.

### 표시 정보

- 최대 5개 케이스 미리보기
- 케이스 키, 이름, 생성일 표시
- **"전체 보기"** 링크로 전체 케이스 목록으로 이동
- **"추가"** 버튼으로 새 케이스 생성

> **참고**: 케이스가 없으면 **"테스트 케이스 만들기"** 버튼이 표시됩니다.

---

## 테스트 스위트 섹션

대시보드 하단에 최근 테스트 스위트를 간략히 보여줍니다.

### 표시 정보

- 최대 5개 스위트 미리보기
- 스위트 이름, 설명, 포함된 케이스 수 표시
- **"전체 보기"** 링크로 전체 스위트 목록으로 이동
- **"추가"** 버튼으로 새 스위트 생성

> **참고**: 스위트가 없으면 **"테스트 스위트 만들기"** 버튼이 표시됩니다.

---

## 온보딩 투어

대시보드를 처음 사용한다면 **"온보딩"** 버튼을 클릭하여 가이드 투어를 시작할 수 있습니다. 투어는 대시보드의 각 영역을 순서대로 안내합니다.

---

## 빠른 작업 가이드

대시보드에서 바로 수행할 수 있는 주요 작업입니다.

| 작업 | 방법 |
|------|------|
| 테스트 케이스 생성 | 케이스 섹션의 **"추가"** 버튼 클릭 |
| 테스트 스위트 생성 | 스위트 섹션의 **"추가"** 버튼 클릭 |
| 테스트 실행 시작 | 테스트 상태 영역의 **"테스트 실행 하기"** 클릭 |
| 프로젝트 URL 공유 | 프로젝트 정보의 **링크 복사** 아이콘 클릭 |
| 전체 케이스 보기 | 케이스 섹션의 **"전체 보기"** 클릭 |
| 전체 스위트 보기 | 스위트 섹션의 **"전체 보기"** 클릭 |
