# 시작하기

Testea에 오신 것을 환영합니다. 이 가이드는 서비스를 처음 사용하는 분들을 위한 안내서입니다.

---

## Testea란?

Testea는 테스트 케이스를 효율적으로 관리하고 실행 결과를 추적할 수 있는 테스트 관리 플랫폼입니다.

### 주요 기능

- **프로젝트 관리**: 프로젝트 단위로 테스트를 관리
- **테스트 케이스**: 테스트 케이스 작성 및 상태 관리
- **테스트 스위트**: 테스트 케이스 그룹화
- **테스트 실행**: 테스트 실행 및 결과 기록
- **마일스톤**: 일정 기반 테스트 계획 관리
- **대시보드**: 테스트 현황 한눈에 파악

---

## 프로젝트 생성하기

### 1단계: 홈페이지 접속

서비스 홈페이지에서 **"무료로 시작하기"** 버튼을 클릭합니다.

### 2단계: 프로젝트 정보 입력

| 필드 | 필수 | 설명 |
|------|------|------|
| 프로젝트 이름 | ✓ | 1~50자, 프로젝트를 식별하는 이름 |
| 식별번호 | ✓ | 8~16자, 프로젝트 접근 비밀번호 |
| 설명 | | 프로젝트에 대한 간단한 설명 |
| 소유자 이름 | | 프로젝트 관리자 이름 |

### 3단계: 프로젝트 생성 완료

생성 버튼을 클릭하면 프로젝트가 생성되고 대시보드로 이동합니다.

> **중요**: 식별번호는 해시 처리되어 저장되므로 분실 시 복구가 불가능합니다. 안전한 곳에 보관하세요.

---

## 프로젝트 접근하기

### 식별번호 입력

1. 프로젝트 URL로 접속합니다. (`/projects/[프로젝트이름]`)
2. 식별번호 입력 화면이 나타납니다.
3. 프로젝트 생성 시 설정한 식별번호를 입력합니다.
4. **"접근하기"** 버튼을 클릭합니다.

### 접근 토큰

- 인증 성공 시 24시간 유효한 접근 토큰이 발급됩니다.
- 토큰이 만료되면 다시 식별번호를 입력해야 합니다.
- 브라우저 쿠키를 삭제하면 재인증이 필요합니다.

### 접근 제한

- 식별번호를 5회 연속 틀리면 15분간 접근이 차단됩니다.
- 차단 해제 후 다시 시도할 수 있습니다.
